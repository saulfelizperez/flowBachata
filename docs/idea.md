# flowBachata

## Problema a resolver

```Bash
Se enfoca en dar informacion sobre sitios de baile, transporte hacia los sitios de baile que se encuentran dentro de BCN y que no pueden ir a sitios que se encuentra en cercanias o lejos debido a falta de coche y clasificacion de escuelas de baile especificas para cada tipo de bachata
```

## Usuario Objetivo

```Bash
Toda persona que este iniciando en el mundo del baile, especificamente de la bachata y quiera tener un marco macro e informacion en un mismo lugar sobre sitios de baile, transporte y comunidades.
```

## Funcionalidades Principales

```Bash
1. Backend (Express + arquitectura por capas)

2. CRUD de tareas

3. Arquitectura por capas

4. Cliente API tipado
```
